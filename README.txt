README
======================
This is a collection of scripts for classifying psychological phenomena in dialogs. It has the following structure:
.
├── annotation_manual: contains annotation manuals
├── training: training scripts
└── inference: sripts for inference


To run the scripts, install the dependencies with:
'''
pip install requirements.txt
'''

If you just want to test the inference, i.e. the prediction, you only need to run the scipts from the 'inference' folder.


For The training scripts are in the 'training folder' Follow the README in that folder to train the models yourself.
